# 📊 Discount Impact Analysis & Revenue Optimization Dashboard

## 🚀 Project Overview

This project focuses on analyzing how discount strategies impact **sales performance, customer behavior, and profitability** using **Python, MySQL, and Power BI**.

The goal is to uncover actionable insights that help businesses **optimize pricing strategies** and maximize revenue while maintaining profitability.

---

## 🎯 Objectives

* Analyze the relationship between **discounts and sales**
* Identify the **optimal discount range** for maximum revenue
* Evaluate **customer and regional behavior**
* Measure the **impact of discounts on profit**
* Build an **interactive dashboard for decision-making**

---

## 🧾 Dataset Description

The dataset contains **12,000+ records** with the following features:

* Order_ID
* Product_Category
* Original_Price
* Discount_Percent
* Quantity_Sold
* Customer_Type
* Region
* Discount_Amount
* Final_Price
* Revenue

---

* **MySQL** (Data Analysis & Querying)

* 📈 Discount vs Sales Trend

## 🧮 Key Insights

* Medium discounts generate the **highest revenue**
* High discounts increase **sales volume but reduce profit**
* Certain product categories respond better to discounts
* Regional differences impact discount effectiveness

---

* Model: **Linear Regression**
* Target Variable: **Revenue**
* Features Used:

  * Original Price
  * Discount Percent
  * Quantity Sold
